# Gleameet Extension

Chrome extension for real-time behavioral coaching during meetings.

## Installation

1. Download [gleameet-extension.zip](https://github.com/rajivchandrasekaran-paintrobot/gleameet-extension/releases/latest/download/gleameet-extension.zip)
2. Unzip the file
3. Go to `chrome://extensions`
4. Enable **Developer mode** (top right toggle)
5. Click **Load unpacked** → select the unzipped folder
